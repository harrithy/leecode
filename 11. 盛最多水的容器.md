# 盛最多水的容器

## 题目描述

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

## 示例

输入：height = [1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

## 思路

- **核心想法**：使用双指针从两端向中间逼近，每次由 **高度较小的一端** 向中间移动指针，过程中维护最大面积。
- **原因**：装水的高度取决于两端较小的那条线（短板效应），同时宽度是两指针之间的距离。
  - 如果移动较高的一侧，宽度变小，高度仍由较矮的一侧决定，面积不可能变大；
  - 只有移动较矮的一侧，才有可能找到更高的线，从而在宽度减小的情况下，面积仍可能增大。

## 算法步骤

1. **初始化**：左指针 `left = 0`，右指针 `right = n - 1`，答案 `maxArea = 0`。
2. **循环** 当 `left < right` 时：
   - 计算当前容器的高度：`h = Math.min(height[left], height[right])`；
   - 计算当前宽度：`w = right - left`；
   - 更新答案：`maxArea = Math.max(maxArea, h * w)`；
   - 如果 `height[left] < height[right]`，左指针右移 `left++`，否则右指针左移 `right--`。
3. 当左右指针相遇时结束循环，返回 `maxArea`。

## 复杂度分析

- **时间复杂度**：整个数组只被双指针各遍历一次，为 \(O(n)\)。
- **空间复杂度**：只使用了常数级额外空间，为 \(O(1)\)。

## 代码实现（JavaScript）

```js
var maxArea = function(height) {
    // 双指针初始化
    let left = 0;
    let right = height.length - 1;
    // 记录最大面积
    let maxArea = 0;

    // 当左指针小于右指针时继续循环
    while (left < right) {
        // 当前容器的高度由较短的那条板决定
        const h = Math.min(height[left], height[right]);
        // 当前容器的宽度是两指针之间的距离
        const w = right - left;
        // 更新最大面积
        maxArea = Math.max(maxArea, h * w);

        // 移动较矮的一侧，尝试寻找更高的板
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }

    return maxArea;
};
```
