# 三数之和

## 题目描述

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

## 示例

```txt
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。
```

```txt
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。
```

```txt
输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。
```

## 提示

- 3 <= nums.length <= 3000
- -10^5 <= nums[i] <= 10^5

## 思路

- **核心想法**：先对数组排序，然后固定一个数 `nums[i]`，在剩余区间用 **双指针** 寻找两数之和等于 `-nums[i]` 的组合。
- **为什么排序**：
  - 排序后可以使用双指针在 O(n) 时间内找到两数之和；
  - 排序后相同的数会相邻，方便跳过重复元素，避免产生重复三元组。
- **去重策略**：
  - 固定数 `nums[i]` 时，如果 `nums[i] == nums[i-1]`，跳过（避免重复枚举相同的第一个数）；
  - 双指针找到一组解后，左右指针都要跳过相同的值。

## 算法步骤

1. **排序**：对数组 `nums` 升序排序。
2. **遍历固定第一个数**：用 `i` 从 `0` 遍历到 `n - 3`：
   - 如果 `nums[i] > 0`，后面不可能有解（三个正数之和不可能为 0），直接结束；
   - 如果 `i > 0 && nums[i] == nums[i-1]`，跳过重复；
3. **双指针寻找另外两个数**：
   - 初始化 `left = i + 1`，`right = n - 1`；
   - 当 `left < right` 时：
     - 计算 `sum = nums[i] + nums[left] + nums[right]`；
     - 若 `sum == 0`，记录结果，然后 `left++`、`right--`，并跳过重复值；
     - 若 `sum < 0`，说明需要更大的数，`left++`；
     - 若 `sum > 0`，说明需要更小的数，`right--`。
4. 返回结果数组。

## 复杂度分析

- **时间复杂度**：排序 \(O(n \log n)\)，双指针遍历 \(O(n^2)\)，总体 \(O(n^2)\)。
- **空间复杂度**：排序需要 \(O(\log n)\) 栈空间（忽略结果数组），总体 \(O(\log n)\)。

## 代码实现（JavaScript）

```js
var threeSum = function(nums) {
    const result = [];
    const n = nums.length;
    
    // 先对数组排序
    nums.sort((a, b) => a - b);
    
    // 遍历数组，固定第一个数
    for (let i = 0; i < n - 2; i++) {
        // 如果第一个数大于 0，后面不可能有解
        if (nums[i] > 0) break;
        
        // 跳过重复的第一个数
        if (i > 0 && nums[i] === nums[i - 1]) continue;
        
        // 双指针
        let left = i + 1;
        let right = n - 1;
        
        while (left < right) {
            const sum = nums[i] + nums[left] + nums[right];
            
            if (sum === 0) {
                // 找到一组解
                result.push([nums[i], nums[left], nums[right]]);
                
                // 跳过重复的左指针值
                while (left < right && nums[left] === nums[left + 1]) left++;
                // 跳过重复的右指针值
                while (left < right && nums[right] === nums[right - 1]) right--;
                
                // 移动双指针
                left++;
                right--;
            } else if (sum < 0) {
                // 和太小，左指针右移
                left++;
            } else {
                // 和太大，右指针左移
                right--;
            }
        }
    }
    
    return result;
};
```

## 总结

三数之和是两数之和的进阶版，关键点：

1. **排序 + 双指针**：将 O(n³) 暴力解法优化到 O(n²)；
2. **去重**：排序后相同元素相邻，遇到重复直接跳过；
3. **剪枝**：当 `nums[i] > 0` 时提前退出，减少无效计算。
