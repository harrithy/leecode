# 字母异位词分组

## 题目描述

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是指由重新排列源单词的字母得到的一个新单词，所有源单词中的字母通常恰好只用一次。

## 示例

**示例 1:**

```txt
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

**示例 2:**

```txt
输入: strs = [""]
输出: [[""]]
```

**示例 3:**

```txt
输入: strs = ["a"]
输出: [["a"]]
```

## 提示

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母

## 解决方案

### 方法

``` javascript

/**
 * @param {string[]} strs
 * @return {string[][]}
 */
function groupAnagrams(strs) {
    // 创建一个哈希表，用于存储字母异位词组
    const map = new Map();
    // 遍历每个字符串
    for (const str of strs) {
        // 创建排序后的键
        const key = str.split('').sort().join('');
        // 检查键是否存在
        if (!map.has(key)) {
            // 创建新数组
            map.set(key, []);
        }
        // 将字符串添加到对应组
        // "eat" "tea" "ate" 做的时候画流程
        map.get(key).push(str);
    }
    // 返回结果
    return Array.from(map.values());
}
// 时间复杂度：O(nk log k)，其中 n 是 strs 的长度，k 是 strs 中字符串的最大长度。
// 空间复杂度：O(nk)
```

## 总结

- str.split('')：将字符串转换为字符数组。例如，'abc'.split('') 返回 ['a', 'b', 'c']。
- arr.sort()：对数组进行排序。例如，['b', 'a', 'c'].sort() 返回 ['a', 'b', 'c']。 // 不仅对数字数组有效，对字符串数组也同样有效
- arr.join('')：将数组转换为字符串。例如，['a', 'b', 'c'].join('') 返回 'abc'。
- map.has(key)：检查哈希表中是否存在某个键。例如，map.has('a') 返回 true 或 false。
- map.set(key, value)：将键值对添加到哈希表中。例如，map.set('a', 1) 将键 'a' 和值 1 添加到哈希表中。
- map.get(key)：获取键对应的值。例如，map.get('a') 返回 1 或 undefined。
- map.values()：返回哈希表中所有值的数组。例如，map.values() 返回 [1, 2, 3]。
- Array.from(map.values())：将哈希表中所有值的数组转换为数组。例如，Array.from(map.values()) 返回 [1, 2, 3]。
