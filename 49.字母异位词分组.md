# 字母异位词分组

## 题目描述

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是指由重新排列源单词的字母得到的一个新单词，所有源单词中的字母通常恰好只用一次。

## 示例

**示例 1:**

```txt
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

**示例 2:**

```txt
输入: strs = [""]
输出: [[""]]
```

**示例 3:**

```txt
输入: strs = ["a"]
输出: [["a"]]
```

## 提示

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母

## 解决方案

### 方法

``` javascript

/**
 * @param {string[]} strs
 * @return {string[][]}
 */
function groupAnagrams(strs) {
    const map = new Map();  // 步骤1：创建哈希表
    
    for (const str of strs) {  // 步骤2：遍历每个字符串
        // 步骤3：创建排序后的键
        const key = str.split('').sort().join('');
        
        // 步骤4：检查键是否存在
        if (!map.has(key)) {
            map.set(key, []);  // 步骤5：创建新数组
        }
        map.get(key).push(str);  // 步骤6：将字符串添加到对应组
    }
    
    return Array.from(map.values());  // 步骤7：返回结果
}
// 时间复杂度：O(nk log k)，其中 n 是 strs 的长度，k 是 strs 中字符串的最大长度。
// 空间复杂度：O(nk)
```

## 总结

- 字母异位词分组问题可以使用哈希表来解决。
- 遍历每个字符串，将排序后的字符串作为键，将原始字符串添加到对应组。
- 最后返回哈希表中的所有值，即字母异位词分组的结果。
